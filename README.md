Serenity Report is an embedded ruby for OpenOffice documents (.odt files). You provide an .odt template with ruby code inside a special markup and the data and Serenity generates the document. If you know erb all of this should sound familiar.

We copied from serenity based https://github.com/kremso/serenity and evolved it.
